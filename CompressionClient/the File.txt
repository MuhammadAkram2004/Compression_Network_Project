--- SECTION 1: Programming ---
C# is a modern, object-oriented programming language developed by Microsoft.
It is widely used for building Windows applications, web services, and games.
Multi-threading in C# allows multiple operations to run concurrently.
The Thread class and Task class are commonly used for this purpose.
TcpListener and TcpClient are used for network communication in C#.
GZipStream provides built-in compression and decompression functionality.

--- SECTION 2: Networking ---
TCP (Transmission Control Protocol) is a connection-oriented protocol.
It guarantees delivery of data packets in the correct order.
A server listens on a specific port for incoming client connections.
Each client connection can be handled in a separate thread for concurrency.
Data is transmitted as raw bytes over the network stream.
File size is sent as a long (8 bytes) before the actual file data.

--- SECTION 3: Compression ---
GZip is a file format and software application for file compression.
It uses the DEFLATE algorithm for compression.
Compression reduces file size, saving bandwidth and storage space.
The compression ratio depends on the type and content of the file.
Text files typically achieve high compression ratios (60-80% reduction).
Binary files like images or videos may not compress as well.
